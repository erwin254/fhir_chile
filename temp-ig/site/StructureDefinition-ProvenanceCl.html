Resource Profile: CL Provenance

Official URL: https://hl7chile.cl/fhir/ig/clcore/StructureDefinition/ProvenanceCl Version: 1.9.3
Active as of 2025-06-02 Computable Name: ProvenanceCL

Copyright/Legal: Usado con el permiso de HL7 International, todos los derechos resevados en los Licencias de HL7 Internacional.

Este Perfil describe la forma de validación respecto a la procedencia de un recurso generado durante un acto clínico.

Usages:

  • Examples for this Profile: Provenance/EjemplValidacionProvCl
  • CapabilityStatements using this Profile: CapabilityStatement Cliente Cl Core and CapabilityStatement Servidor Cl Core
