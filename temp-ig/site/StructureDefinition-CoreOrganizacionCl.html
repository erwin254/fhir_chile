Resource Profile: CL Organización

Official URL: https://hl7chile.cl/fhir/ig/clcore/StructureDefinition/CoreOrganizacionCl Version: 1.9.3
Active as of 2025-06-02 Computable Name: OrganizacionCL

Copyright/Legal: Usado con el permiso de HL7 International, todos los derechos resevados en los Licencias de HL7 Internacional.

Este Perfil ha sido desarrollado para cubrir las necesidades del Caso de Uso de Receta Electrónica. Sin embargo, se ha modelado para también cubrir las necesidades nacionales de un Recurso de Organización en Salud

Usages:

  • Refer to this Profile: CL Auditoria, CL Localización, CL Medicamento, CL Observación... Show 7 more, CL Paciente, CL RolClinico, CL Documento, CL Encuentro Clínico, CL Inmunización, CL Provenance and CL Signos Vitales
  • Examples for this Profile: Laboratorio ACME de Chile and Centro de Salud Familiar Arauco
  • CapabilityStatements using this Profile: CapabilityStatement Cliente Cl Core and CapabilityStatement Servidor Cl Core
